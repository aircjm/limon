# limon

limon 是一款极易搭建的个人笔记管理服务。


## 项目愿景

limon 项目旨在打造一个以最简便的方式搭建简单、稳定和可扩展的个人笔记管理服务。后续计划扩展成GTD和卡片笔记服务一体，形成个人时间管理，个人笔记管理一体的信息管理服务。

由于使用markdown 所以只能使用特定的模板 Markdown

## 主要特性

- 导入trello的笔记
- 笔记内容转换成markdown的anki的note方便复习回顾





## 注意事项
- 需要指定对应的DB数据库
